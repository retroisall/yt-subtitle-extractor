B — 全局 Toggle 解析

狀態 1 — 兩個都關閉
第 2 / 42 行
兩個功能皆關閉。Focus 列僅高亮，YouTube 不會自動動作。
狀態 2 — 同步播放 ON
第 2 列 第 2 / 42 行
每次 focus 換到新的一列，YouTube 自動跳轉並播放該列時間點（單次觸發，不重複）。
狀態 3 — 循環此句 ON
第 2 列 · 循環中 第 2 / 42 行
Focus 停在哪一列，YouTube 就持續重播那一列的聲音。切換 focus → 新的一列開始循環，前一列停止。
狀態 4 — 兩個都 ON（同步 + 循環）
第 2 列 · 循環中 第 2 / 42 行
最強學習模式：跳到這列 → 自動從時間點開始播 → 持續循環。換列時無縫切換到下一句。
Focus 切換到任一列時，自動通知 YouTube 分頁跳轉到該列 startTime 並播放。

單次觸發 不會重播，只在 focus 改變時執行一次。
Focus 停留在哪一列，YouTube 就持續重播 startTime → endTime。

切換 focus → 舊列停止，新列開始循環。
循環期間可自由編輯文字，聲音不中斷。
Tab / 點擊換到新列 → 自動跳轉時間點 → 開始循環聲音。
使用者可一邊聽 → 一邊對照 → 一邊修改字幕，全程不用碰滑鼠控制影片。
