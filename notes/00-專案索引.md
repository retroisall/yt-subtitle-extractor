# Chrome 字幕套件 — 專案索引

## 專案概覽

> Chrome MV3 擴充套件，功能涵蓋 YouTube 字幕提取、即時翻譯、單字本、字幕編輯器與詞彙儀表板。

- **程式碼總量**：~16,400 行（JS + HTML + CSS）
- **架構**：Manifest V3 / Service Worker / Content Script / Main World
- **外部服務**：Firebase / Firestore / Google Translate API / LINE Bot

## 執行環境三層架構

```
┌─────────────────────────────────────────┐
│  Page Context (Main World)              │
│  [[patch]]  [[inject]]                  │
│  • 存取 window.ytInitialPlayerResponse  │
│  • 無法使用 chrome.* API                │
├─────────────────────────────────────────┤
│  Isolated World (Content Script)        │
│  [[content]]  [[editor]]  [[vocab-dashboard]] │
│  • chrome.* API 完整存取                │
│  • 無法直接讀 YT 全域變數               │
├─────────────────────────────────────────┤
│  Service Worker                         │
│  [[background]]  [[firebase]]           │
│  • Firebase / Firestore 操作            │
│  • 無 DOM 存取、會被終止重啟            │
└─────────────────────────────────────────┘
```
