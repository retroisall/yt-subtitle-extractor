安裝方式

三步驟，60 秒搞定

不需要帳號，不需要 API key，裝好就能用。

01

前往 Chrome 商店

搜尋「學習Bar」，點「加入 Chrome」，一鍵安裝，不需要帳號。

02

開啟 YouTube

前往任意 YouTube 影片，右側自動出現「學習Bar」側邊欄，選擇字幕語言即可。

03

開始學習

點字幕上的單字查字典、右鍵儲存生字、點字幕背景啟動單句循環。就這樣。
