YT Subtitle Demo - 安裝說明
============================
 
1. 開啟 Chrome 瀏覽器
2. 網址列輸入：chrome://extensions/
3. 右上角開啟「開發人員模式」
4. 點擊「載入未封裝項目」
5. 選擇這個資料夾（yt-subtitle-demo）
6. 開啟任意 YouTube 影片頁面
7. 右側會出現「字幕提取器」側邊欄
8. 點擊語言按鈕載入字幕
9. 點擊字幕可跳轉到該時間點
 
注意事項：
- 僅在 youtube.com/watch 頁面生效
- 需要影片本身有字幕（手動或自動生成皆可）
- 切換影片時會自動重新載入字幕清單
