安裝方式

三步驟，60 秒搞定

不需要帳號，不需要 API key，裝好就能用。

01

下載套件

點下方按鈕，從 GitHub 下載 ZIP 並解壓縮到任意資料夾。

02

載入 Chrome

開啟 chrome://extensions，開啟開發人員模式，點「載入未封裝項目」選資料夾。

03

開啟 YouTube

前往任意 YouTube 影片，右側自動出現「學習Bar」側邊欄，選擇語言即可開始。

04

開始學習

點擊字幕上的單字查字典、右鍵儲存生字、點背景啟動循環。就這樣。
